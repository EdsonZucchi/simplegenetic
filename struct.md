# Estrutura algoritmo genético

## Definir o genoma e seus genes

## Inicialização da população
- Se atentar ao tamanho da população

## Avaliação dos indivíduos
- Fitness function, penalizar e beneficiar com 
base nos dados da população

## Seleção dos indivíduos
- Select function, utilizado método de roleta com classificação

## Cross-over
- Utilizar os genes dos pais como base para gerar os filhos

## Mutação
- Verifica se vai ser realizado a mutação
- Cuidar para não extrapolar o limite

## Nova geração
- Seleção -> Crossover -> Mutação
- Nova geração manter os melhores (Elitismo)

## Termina
- Atingir uma meta, X gerações baterem essa meta
- Sem progresso, X gerações sem progresso para a aplicação
